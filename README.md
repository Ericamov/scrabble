# CS50 – Scrabble

## Description

This program compares two words by calculating their Scrabble scores and determines which player has the higher score.

## How it works

The program:

1. Prompts two players to enter a word each
2. Converts each letter to its corresponding Scrabble point value
3. Sums the total score for each word
4. Compares the scores and declares the winner (or a tie)

## Usage

```
make scrabble
./scrabble
```

## Example

```
Player 1: COMPUTER
Player 2: SCIENCE
Player 1 wins!
```
